# Stevens-HTML-and-CSS

## Running locally

Use npm script (Codespaces or local machine):

- `npm run http-serve` (recommended) to launch on `0.0.0.0:8000`
- `npm run serve` (existing command) also works

### GitHub Codespaces ports

In Codespaces, ensure port `8000` is forwarded in the Ports panel and open the forwarded URL (or use `https://8000-<workspace>-<hash>.githubpreview.dev`).
